# BIBLIAPPS Explorer

Um emulador de explorador de arquivos minimalista e funcional, desenvolvido para praticar manipulação de DOM e layouts modernos.

## 🚀 Tecnologias
- **HTML5** (Estrutura semântica)
- **CSS3** (Flexbox & Grid Layout)
- **JavaScript ES6+** (Renderização dinâmica)
- **FontAwesome** (Iconografia)

## 🛠️ Funcionalidades
- [x] Renderização dinâmica de arquivos a partir de um objeto JSON.
- [x] Busca em tempo real.
- [x] Layout responsivo (Grid adaptável).
- [x] Interface limpa estilo macOS/Windows.

## 📁 Como usar
Basta abrir o arquivo `index.html` no seu navegador ou utilizar o Live Server no VS Code.
